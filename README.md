# FinStud

FinStud is a personal finance tracker with a static HTML/CSS/JS frontend and a Node.js/Express + MySQL backend. It lets users sign up, track income and expenses, manage assets and liabilities, set financial goals, and view a dashboard summary.

## Project Structure

```
finstud-project/
├── finstud/            # Frontend (static HTML/CSS/JS)
│   ├── css/
│   ├── js/
│   └── pages/
└── finstud-backend/     # Backend (Node.js + Express + MySQL)
    ├── config/
    ├── middleware/
    └── routes/
```

## Getting Started

See [`finstud-backend/README.md`](./finstud-backend/README.md) for full setup instructions (installing dependencies, configuring the database, environment variables, and running the server).

## Tech Stack

- **Frontend:** HTML, CSS, vanilla JavaScript
- **Backend:** Node.js, Express, JWT auth, Nodemailer
- **Database:** MySQL
